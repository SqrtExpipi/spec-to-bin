# Changelog

All notable changes to Spec to BIN are documented in this file.

The project follows [Semantic Versioning](https://semver.org/). The app version and the
`formatVersion` in a template are separate: app `v0.2.0` continues to use template format
`0.1`.

## [0.2.0] - 2026-07-23

### Added

- Detailed English and Japanese user guides.
- JSON drag-and-drop, first-run choices, and a directly accessible JSON editor.
- Batch row selection, an in-editor field clipboard, repeated-row expansion, and fixed-length
  string test-value generation.
- Comparison between generated bytes and an existing BIN, including SHA-256 hashes and
  field-relative difference locations.
- Restore-to-saved and new-template actions.
- Persistent JSON syntax errors with line and column information.

### Changed

- Distinguished the active row from batch-selected rows and kept batch actions visible while
  scrolling long templates.
- Made paste insert below the active row and made batch deletion undoable.
- Refined the Hex preview, responsive layout, action priority, validation feedback, help text,
  and English/Japanese labels.
- Improved the self-contained offline build and release verification.
- Expanded metadata and static pages so the project and its supported workflows are easier to
  discover.

### Removed

- Removed the test-data package ZIP export. BIN and JSON remain explicit, separate exports.

## [0.1.0] - 2026-07-12

### Added

- Initial public release.
- Structured JSON template loading and saving.
- GUI field editing, validation, Hex preview, copy formats, and BIN export.
- English and Japanese UI, light/dark/system themes, PWA support, and an offline release.
