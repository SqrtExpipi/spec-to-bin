## Longer announcement / Japanese

```text
仕様書ベースでテスト用バイナリファイルを作成するOSS「Spec to BIN」を公開しました。

JSONでバイナリ構造を定義し、GUIで値を編集、Offsetや入力値を検証して、Hexを確認しながら.binとして保存できます。通信電文、組込み設定データ、EEPROMデータ、テストペイロードの作成を想定しています。

日本語/英語、ASCII/UTF-8/Shift_JIS、ライト/ダークテーマ、PWA、単一HTMLのオフライン版に対応しています。データはブラウザ内で処理し、アップロードやテレメトリは行いません。

https://sqrtexpipi.github.io/spec-to-bin/
```

## Longer announcement / English

```text
I released Spec to BIN, an open-source browser tool for generating test binary files from structured JSON templates.

Define binary fields, edit values in a table, validate offsets and ranges, preview generated Hex, and export a .bin file. It supports English and Japanese, ASCII/UTF-8/Shift_JIS, light and dark themes, PWA installation, and a self-contained offline build.

Processing stays in the browser with no upload or telemetry.

https://sqrtexpipi.github.io/spec-to-bin/
```
