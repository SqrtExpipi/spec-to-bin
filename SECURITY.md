# Security policy

Spec to BIN processes templates and generated binary data locally in the browser. It does not intentionally upload template data, generated bytes, or telemetry.

## Reporting a vulnerability

Please use GitHub private vulnerability reporting for this repository. Do not include customer specifications, credentials, production binary data, or other confidential material in a public issue.

## Supported version

Until the first stable release, only the latest version on `main` is supported.
